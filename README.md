# 🤖 Inteligências Artificiais Generativas

Repositório criado para documentar minha jornada de aprendizado em Python, Inteligência Artificial Generativa, Engenharia de Prompt e desenvolvimento de Chatbots.

Ao longo deste projeto, desenvolvi exercícios práticos, desafios de lógica, automações simples e atividades voltadas para o uso de ferramentas de IA no desenvolvimento de soluções computacionais.

## 🎯 Objetivo

O objetivo deste repositório é reunir atividades realizadas durante os estudos de Inteligência Artificial Generativa, fortalecendo conhecimentos em:

* Programação com Python
* Lógica de programação
* Estruturas condicionais e de repetição
* Manipulação de dados
* Engenharia de Prompt
* Desenvolvimento de Chatbots
* Aplicações práticas de Inteligência Artificial
* integração de API
